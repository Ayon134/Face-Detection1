Just test---------
1) Just run the faces.py file
2) camera will start running and you can try testing by showing it some pictures
3) trained model can only recognize from the folder images

Customize-----------
1) First go to recognizers folder and delete the .yml file
2) Make folders of your charecters in image folder and put few images of your charecters in respective named folders.
3) Run the faces-train.py (may take a while)
4) Finally run faces.py and again test.
5) You may change numarical values for better result.
